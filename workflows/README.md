# Workflow Overview

Financial Operations Assistant is implemented as a functional Progressive Web App (PWA) powered by Anthropic Claude.

The application uses Claude Managed Agents and the Claude API as its AI runtime.

## Main Flow

text User interaction → conversational financial input → Anthropic Claude Managed Agent → AI-powered interpretation and response generation → document attachment when required → transaction classification → structured financial registration → reporting-oriented records 

## Functional Scope

The current MVP supports:

- Conversational registration of income and expenses
- AI-assisted interpretation of financial inputs
- Document attachment
- Structured transaction capture
- Financial data organization
- Reporting-oriented records
- Mobile installation as a PWA
